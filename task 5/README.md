# Task 5 – AVR Control Flow (RJMP, JMP, IJMP)

## 🎯 Objective
Demonstrate different control flow instructions:
- `RJMP`
- `JMP`
- `IJMP`

---

## ⚙️ Implementation

### Phase 1 – RJMP
- Polling loop waiting for button press

### Phase 2 – JMP
- Jump to selected mode:
  - Mode A → slow blink
  - Mode B → double blink
  - Mode C → fast strobe

### Phase 3 – IJMP
- Jump table for runtime action selection

---

## 📊 Behavior

- Button press selects mode
- Further presses change behavior dynamically
